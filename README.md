## ⚡ Cara Menjalankan

### 1. Pastikan Python sudah terinstall
Cek versi Python yang terpasang:
```bash
python --version
```
Belum punya Python? Unduh di [python.org](https://www.python.org/) →

### 2. Clone repositori ini
```bash
git clone https://github.com/ferdymf/Python.git
cd Python
```

### 3. Jalankan proyek yang diinginkan
```bash
python nama_proyek.py
```

**Contoh:**
```bash
python Pong_Game.py
python Kecocokan_cinta.py
```
